Analytic “AND” Function

By mapping positive values to True and negative values to False, we explore the mathematical construction of an "AND" function. While a smooth "AND" function is achievable, we demonstrate that generating an analytic one is fundamentally impossible, and delve deeply into the implications of this result.

Introduction

While working on a different problem (Self-similar PDEs), I wanted to terminate a ODE solver if it crossed some boundary 1 or some boundary 2. Usually I would just use some root finding algorithm to find a crossing, it led me to think I should construct some function with is negative if I have crossed either of the boundaries, and positive iff I haven't crossed either of the boundaries. How hard can it be?

At first glance, just using a "MIN" function works, it essentially encodes an "AND" function if positive was TRUE and negative was FALSE. However since many root finding algorithms (such as Newton's method) relay on using the derivative we should try to construct a "nicer" function. As it turns out, we can construct a smooth "AND" function but it is impossible to do such for an analytic function.

Problem Statement

We call a function $f:\mathbb{R}^2 \to \mathbb{R}$ an "AND" function if:

  • if $x>0$ and $y>0$, then $f(x,y)>0$
  • if $x<0$ or $y<0$, then $f(x,y)<0$

As we mentioned earlier, the simplest way is to just use a "MIN" function, which is continuous everywhere. and anywhere where $x \neq y$ it also smooth and analytic.

We can also construct a "AND" function which is smooth everywhere, for example: Let $$ \phi(t) = \begin{cases} e^{-1/t} & \text{if } t > 0 \\ 0 & \text{if } t \le 0 \end{cases} $$ then we can define: $$ f(x,y) = \phi(x)\phi(y) - \phi(-x) - \phi(-y) $$ which is smooth and satisfies the "AND" condition.

Non-analyticity of "AND"

Constructing an analytic "AND" function is impossible, We will now show a formal proof, and later we will discuss what it means.

  1. if $f(x)$ is analytic, and $\forall x \in (a,b) f(x) = 0$, then $f(x) = 0$ for all $x \in \mathbb{R}$.
  2. Let us now look at line $y=0$ and define $g(x) = f(x,0)$, when $x>0$, for every $\epsilon>0$ $f(x,\epsilon) > 0$ and $f(x,-\epsilon) < 0$, by continuity of $f$ we have $f(x,0) = g(x) = 0$ for every $x>0$.
  3. When $x<0$ , then by the AND condition we have $f(x,0) = g(x) < 0$.
  4. Since $f(x,y)$ is analytic, then so is $g(x)$
  5. By 1 and 2, we have $g(x) = 0$ for all $x \in \mathbb{R}$, but this is a contradiction to 3

Functional complete set

we notice that Constructing an analytic "NOT" gate is pretty trivial: $$ f(x) = -x $$ So if we had an analytic "AND" function, we could construct any logical function we wanted as a composition of "AND" and "NOT" functions. This intiution increases as we notice that a "XOR" analytic function is very easy to construct: $$ f(x,y) = xy $$ and it feels like this is not a coincidence, since "XOR" and "NOT" are not enough to construct a Functional complete set.
