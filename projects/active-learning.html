Active Learning

Query strategies and sample-complexity bounds for learning from adaptively chosen labelled examples.

Setting

In the standard PAC learning model a learner receives $m$ labelled examples drawn i.i.d. from an unknown distribution $\mathcal{D}$ and must output a hypothesis $h$ with small generalisation error $\mathrm{err}(h)$. Active learning relaxes the i.i.d. assumption: the learner may choose which points to query an oracle for labels, aiming to achieve the same error with far fewer labels.

Version spaces and query-by-disagreement

Given a hypothesis class $\mathcal{H}$ and a set of unlabelled points, define the version space $V_S = \{h \in \mathcal{H} : h \text{ consistent with } S\}$. The query-by-committee strategy labels a point $x$ when the committee of hypotheses disagrees:

$$x^* = \arg\max_{x \in U}\; \mathrm{H}\!\left[\mathbb{P}_{h \sim V_S}[h(x)=1]\right]$$

where $\mathrm{H}[\cdot]$ is the binary entropy. The disagreement coefficient $\theta$ of $\mathcal{H}$ under $\mathcal{D}$ governs the speed-up over passive learning.

Sample complexity

For a hypothesis class of VC dimension $d$, passive learning requires $\tilde{O}(d/\varepsilon)$ labels to achieve error $\varepsilon$. Under favourable geometry, active learning can reduce this to $\tilde{O}(d \log(1/\varepsilon))$ — an exponential improvement in $1/\varepsilon$.

$$m_{\text{active}} = \tilde{O}\!\left(\theta^2 d \log\frac{1}{\varepsilon}\right) \quad \text{vs.} \quad m_{\text{passive}} = \tilde{O}\!\left(\frac{d}{\varepsilon}\right)$$

Uncertainty sampling

The simplest practical strategy is uncertainty sampling: always query the unlabelled point on which the current model is least confident. For a probabilistic classifier $p_\theta(y|x)$,

$$x^* = \arg\min_{x \in U}\; \max_y\; p_\theta(y \mid x)$$

Despite its simplicity, theoretical guarantees are weaker than for disagreement-based methods; this gap is a focus of ongoing work.
